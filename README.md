## Exploratory Data Analysis (EDA)
This project is a part of my CodSoft Data Analytics Internship.

### Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand its structure, identify patterns, analyze relationships, detect outliers, and generate meaningful insights.

### Tasks Performed
- Loaded the Titanic dataset using Pandas.
- Examined the dataset using `head()`, `info()`, and `describe()`.
- Analyzed the distribution of numerical features.
- Identified relationships between variables using correlation analysis.
- Detected outliers using box plots.
- Summarized key findings from the dataset.

### Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook (VS Code)

## Dataset
Titanic Dataset
